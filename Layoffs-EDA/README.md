# # Layoffs Exploratory Data Analysis Project

## Objective
Analyze layoffs trends across companies, industries, and time.

## Key Analysis Performed
- Identified companies with the highest layoffs
- Analyzed layoffs trends over time (yearly & monthly)
- Compared layoffs across countries
- Calculated rolling totals to track trends

## SQL Techniques Used
- Aggregations (SUM, AVG)
- GROUP BY
- Window Functions (rolling totals)
- Date-based analysis

## Key Insights
- Layoffs are highly concentrated among a small number of large companies  
- Significant spikes in layoffs align with specific time periods, indicating potential economic downturns  
- Patterns suggest industry-wide contractions during certain periods

## Business Value
This analysis provides visibility into workforce reduction trends, helping organizations and stakeholders understand economic cycles, industry contractions, and potential future risks.

## Key Outcome
Provided a data-driven overview of layoffs trends, highlighting patterns useful for strategic workforce and economic analysis.
